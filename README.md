# Code and metadata for Systematic Literatyre Review

This repository provides supporting code and bibliographic metadata for the article:

Guerrero-Domínguez, D., Lima Azevedo, C., Karlsson, S., and Nielsen, O. A. (2026).  
*Quantum Computing for Predictive Transport Modelling: A Systematic Literature Review and Future Research Needs.*  
IEEE Transactions on Intelligent Transportation Systems.  
https://doi.org/10.1109/TITS.2026.3676854

The materials in this repository are intended solely to support transparency and reproducibility of the literature review presented in the paper.

---

## Contents

### `ArticlesData/`
RIS files exported from:
- Scopus  
- Google Scholar  
- DTU FindIt  

These files contain only bibliographic metadata (titles, authors, abstracts, DOIs).  
No copyrighted full‑text articles are included.

### ReviewCode.ipynb
A simple Python notebook used to parse RIS files, clean and filter metadata, and remove duplicates.  
The code is minimal and provided for methodological transparency.
